# Fork tutorials

These tutorials use the live [Fork web application](https://forkdynamics.com). Each tutorial gives step-by-step instructions and screenshots. You do not need prior knowledge of Fork or dynamical systems.

## Tutorials

### [Continue a fold bifurcation in a one-variable ODE](fold-bifurcation/README.md)

Create the equation $\dot{x}=\mu-x^2$. Learn the main parts of the Fork window. Calculate an orbit and an equilibrium. Continue the equilibrium through a fold bifurcation. Then, compare stable and unstable equilibria with nearby orbits.

### [Logistic map: from one orbit to the period-doubling route to chaos](logistic-map-period-doubling/README.md)

Open the existing **LogisticMap** system. Calculate a chaotic orbit and a stable fixed point. Make a bifurcation diagram. Continue cycles with periods 1, 2, 4, 8, 16, and 32. Then, compare the period-doubling points with the Feigenbaum constant.

### [Rössler flow: from a stable equilibrium to four period doublings](rossler-period-doubling-route/README.md)

Open the existing **Rossler** system. Show an orbit in a three-dimensional State Space Scene. Make an Event Map with a cobweb. Continue a stable equilibrium through a Hopf bifurcation. Then, continue stable limit cycles through four period-doubling bifurcations and compare each cycle with a matching orbit.

## Suggested order

Start with the fold-bifurcation tutorial if you are new to Fork. Next, use the logistic-map tutorial to learn period doubling in a one-variable map. Use the Rössler tutorial last because it combines three-dimensional views, an Event Map, and limit-cycle continuation.
